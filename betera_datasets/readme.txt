Условия
Выгрузить игроков, которые:     
С 12:00 14.03.2022 сделали ставку в пре-матч на события раздела «Киберспорт».
Минимальная сумма ставки составляет 10 BYN с коэффициентом не менее 1,5, в экспрессе все события на киберспорт с кф каждого события от 1.50
Ставка должна быть рассчитана не позднее 12:00 15.03.2022.
Ставки вида «система» не учитываются.
Ставки CashOut, возвраты и FreeBet не участвуют в Бонусе.

Необходимо решение в двух форматах:
1. Только средством SQL - файлы есть таблицы
2. Только средством Python
